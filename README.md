# Maze Game - Maze in p5.js

A simple maze game written in JavaScript using the p5.js library.  
The player controls a red circle through a randomly generated maze and must reach the red goal square.

## Features

- Randomly generated maze (DFS algorithm)
- Controls with arrow keys
- Move counter
- Win notification when reaching the goal

## How to run?

1. Open the project in VS Code or any code editor.
2. Run a local server (for example, the **Live Server** extension in VS Code).
3. Open `index.html` in your browser.
4. Use the arrow keys to move the player.

## File structure

- `index.html` — main HTML file loading p5.js and the game script
- `sketch.js` — game logic and maze generation
- `style.css` — page styling

## Technologies used

- JavaScript
- p5.js
- HTML5 & CSS3
